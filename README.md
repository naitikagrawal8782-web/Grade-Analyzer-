# Student Grade Analyzer – Introduction to Programming Project

# Overview
Student Grade Analyzer is a console-based program created for the  
Introduction to Problem Solving & Programming course.  
It stores student marks, calculates their average, assigns grades,
and identifies the top-performing student.

This project follows all required steps:
- Problem Definition
- Requirements
- Top-Down Design
- Algorithms
- Implementation
- Testing

# Features
- Add student name & marks for 3 subjects  
- Automatic average calculation  
- Automatic grade assignment (A, B, C, D)  
- View all student records  
- Identify top student  
- Simple menu interface  

# Technology Used
- Python (basic programming concepts)

# How to Run
1. Open terminal / CMD
2. Go to the folder where file is saved
3. Run:
python student_grade_analyzer.py
